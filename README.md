# Automation Project - Playwright

Proyecto de automatización de pruebas usando Playwright con Python.

## Descripción

Este proyecto contiene pruebas automatizadas usando Playwright para la automatización de navegadores.

## Requisitos

- Python 3.8+
- Playwright

## Instalación

1. Instalar las dependencias:
```bash
pip install -r requirements.txt
```

2. Instalar los navegadores de Playwright:
```bash
playwright install
```

## Uso

Ejecutar las pruebas:
```bash
pytest tes_playwrightbasic.py
```

## Estructura del Proyecto

- `tes_playwrightbasic.py` - Pruebas básicas de Playwright
